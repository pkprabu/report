3. Step-by-Step Workflow

Step 1: Open the Tool
Open the `index.html` file in your browser.

Step 2: Build the Report
Use the **Bulk Campaign Creator** to paste lists of names and URLs, select a target section, and click **"Add Campaigns"**. Repeat for each section as needed.

Step 3: Fine-Tune and Edit
Review the generated report. Click on any cell (metrics, campaign names, section headers) to make manual edits. Use the **Layout Controls** to add or delete single rows if necessary.

Step 4: Copy for Outlook
Once the entire report is perfect, click the green **"Copy Report for Outlook"** button. This copies the final, clean HTML to your clipboard.

Step 5: Paste into Email
Create a new email in Outlook and simply **Paste** (`Ctrl+V` or `Cmd+V`) into the message body. The report will appear perfectly formatted.
